Obtain and Use API Token

You can obtain an API token for the secured methods here.
The obtained token can be used in a bearer authorization header like this: 'Authorization: Bearer [token]'
To use the token in the api page, you must click 'Authorize' and fill in the apiKey field with the following: 'Bearer [token]'.
